# Framework Reference Card

Every framework, its core question, its formula, and where it lives in the skill flow.

| # | Framework | Core Question | Formula / Structure | Skill |
|---|---|---|---|---|
| 1 | **6P Story** | Who is this user, really? | Person → Problem → Pursuit → Pitfall → Pivot → Payoff | Skill 1 |
| 2 | **BMAP** | Which behavioral variable is broken? | B = M × A × P (Motivation × Ability × Prompt) | Skill 2 |
| 3 | **NPV / Psych Framework** | Where does the psych budget go negative? | NPV = Motivation − Friction (per touchpoint) | Skill 3 |
| 4 | **B.I.A.S. Loop** | What's failing in the perception-to-action chain? | Block → Interpret → Act → Store (sequential) | Skills 4–7 |
| 5 | **DPP** | Is this screen asking too much? | Count all decisions on one page. >3–4 = red flag | Skill 6 |
| 6 | **Kano Model** | What type of feature is this? | Must-have / Performance / Delighter (decay over time) | Skill 7 |
| 7 | **Journey Map** | Where are the emotional highs and lows? | Psych Level (Y) × Customer Journey (X), 5–6 moments | Skill 8 |
| 8 | **Reorder Matrix** | Can we fix this by changing the sequence? | Pit→Peak swap, Jump insertion, Transition shortening | Skill 9 |
| 9 | **B.I.A.S. for Relationships** | How do I communicate this to stakeholders? | Block→Interpret→Act→Store applied to the audience | Skill 10 |
| 10 | **3-Step Feedback Formula** | How do I handle design pushback? | Acknowledge → Explain reasoning → Bounded alternative | Skill 10 |
| 11 | **Regret Test** | Would an informed user still choose this? | If yes → ethical. If no → manipulation. | Skill 11 |
| 12 | **Manipulation Matrix** | Who does this design serve? | 2×2: User benefit × Maker uses product | Skill 11 |
| 13 | **Black Mirror Test** | What breaks at scale over time? | Scale to 1B users, 10 years. What goes wrong? | Skill 11 |
| 14 | **Sequencing Rule** | Did we fix the right level? | Upstream phase must be resolved before downstream | Skill 12 |
| 15 | **GEQ (General Empathy Questions)** | What does research input look like? | Hope? / Frustration? / Barrier? | Skill 1 |
| 16 | **SEQ (Specific Empathy Questions)** | How do we go deeper? | Task-specific follow-ups from GEQ responses | Skill 1 |
| 17 | **Activation Threshold** | Will a prompt work here? | Trade-off curve on BMAP graph. Above = yes. | Skill 2 |
| 18 | **Humane Notification Reset** | Is this notification for the user or for us? | Need it? Want it? Serves user or metric? | Skill 11 |

### Framework Selection Logic

```
User has a product problem
├── Don't know who the user is → Skill 1 (6P Story) + Skill 2 (BMAP)
├── Know the user, need to find the problem → Skill 3 (NPV) → Skills 4-7 (B.I.A.S.)
├── Know the problem, need to fix it → Skill 8 (Journey Map) → Skill 9 (Improve)
├── Have the fix, need buy-in → Skill 10 (Communicate)
├── Have buy-in, need ethics check → Skill 11 (Ethics)
└── Shipped, need to validate → Skill 12 (Iterate)
```
